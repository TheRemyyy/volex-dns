# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-12-29

### Added
- **Initial Release**
- **Race Mode**: Parallel querying of multiple DNS-over-TLS (DoT) upstreams.
- **In-Memory Cache**: Concurrent-safe TTL-based caching for faster subsequent lookups.
- **DNS-over-TLS (DoT)**: Secure and encrypted communication with upstream providers.
- **Local PTR Handling**: Custom responses for local 127.0.0.1 PTR requests.
- **JSON Configuration**: Flexible configuration via `config.json`.
- **Documentation**: Detailed architecture, configuration, and development guides.
